HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user.
  • "ul" is an unordered list while "ol" is an ordered list with "li" being line items for both
  • Class and id are identifiers that CSS and JavaScript can point to

CSS

  • A margin indicates how much spaces we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • You can set the style of multiple elements that will use the paramaters by using a ",".
  • Make sure your CSS file is linked to your html file or nothing will change

Git

  • Before beginning to code always do a pull of your main to make sure you're not working with outdated code
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • if we don't git push our commits won't get sent to the the cloud version of our branch.

JavaSCript

  • A variable is a named contatiner that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Functions can help save a lot of space when needing to call blocks of code multiple times.
  • Similarly, for loops and if/else statements can cut down on the amount of code that needs to be written.
